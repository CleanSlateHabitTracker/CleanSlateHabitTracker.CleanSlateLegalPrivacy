# Clean Slate – Habit Tracker

Clean Slate is a simple, focused habit tracking app designed to help users build better routines and stay consistent.

## Features

- Track daily habits
- Monitor streaks
- Clean, distraction-free interface
- Optional Pro subscription for enhanced features

## Subscriptions

Clean Slate offers an optional auto-renewable subscription ("Clean Slate Pro") which unlocks additional functionality.

Subscriptions are processed securely through Apple’s App Store.

## Privacy
